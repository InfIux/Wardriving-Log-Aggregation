# Wardriving-Log-Aggregation
I wanted to create a script in Python to not only aid in my learning of coding, but also to make aggregating one or more log files into a single large file for uploading WDGoWars and or WiGLE.

As of right now, I wardrive with a Marauder v8. Wardriving logs are stored as a .log file, with the first two lines being labeling and description, and the rest is captured data.
You could just copy and paste everything file by file, but that isn't super fun. So why not create a Python script for it.
In this script, I am using the "tkinter" module for easy file selection instead of having to always change the script for every new log file introduced.

Once I finish my C5 wardriving cluster rig, I probably won't even need this anymore, but hopefully someone else finds it and makes use of it!
